# Modelo base para la web: Marmoleria Villalobos

## 1. Proposito del modelo
Este documento sirve como modelo de referencia para construir la pagina web de Marmoleria Villalobos. Resume la arquitectura visual, la estructura de contenido y el tono de marca.

## 2. Concepto visual
- Estilo: premium industrial
- Sensacion: firme, elegante, artesanal
- Base cromatica:
  - Blanco para limpieza visual
  - Negro para contraste y profundidad
  - Naranja para enfasis y conversion

## 3. Mapa del sitio
1. Inicio
2. Servicios
3. Materiales
4. Galeria
5. Nosotros
6. Contacto

## 4. Modelo de layout de una pagina principal

### Header
- Logo a la izquierda
- Navegacion centrada o a la derecha
- Boton destacado: Cotiza ahora

### Hero
- Frase principal de alto impacto
- Texto breve de apoyo
- Dos botones de accion
- Imagen o fondo con textura de piedra

### Bloque de confianza
- Numeros o beneficios rapidos
- Ejemplos:
  - Atencion personalizada
  - Acabados de precision
  - Proyectos residenciales y comerciales

### Servicios destacados
- Grid de 3 a 6 tarjetas
- Cada tarjeta con titulo, descripcion corta e icono o imagen

### Materiales
- Tarjetas por tipo de material
- Cada una con nombre, apariencia y uso sugerido

### Galeria de proyectos
- Rejilla visual adaptable
- Espacio para imagen, tipo de proyecto y material usado

### Nosotros o diferencial
- Texto corto sobre experiencia, calidad y compromiso
- Lista de diferenciales concretos

### Contacto
- Datos directos del negocio
- Formulario corto
- Boton de WhatsApp visible

### Footer
- Logo
- Resumen de servicios
- Datos de contacto
- Redes o enlaces rapidos

## 5. Modelo de tono de comunicacion
- Profesional pero cercano
- Seguro y directo
- Sin tecnicismos innecesarios
- Enfocado en calidad, durabilidad y detalle

## 6. Modelo de mensajes clave
- Trabajos en marmol y granito con acabados de alta calidad
- Soluciones personalizadas para hogar, negocio y proyectos especiales
- Precision, experiencia y materiales que perduran
- Cotizaciones claras y atencion directa

## 7. Modelo de CTA
- Solicitar cotizacion
- Ver proyectos
- Hablar por WhatsApp
- Agendar asesoria

## 8. Modelo de componentes visuales
- Botones primarios en naranja
- Tarjetas blancas con detalles oscuros
- Secciones oscuras con texto claro para contraste
- Lineas divisorias sutiles
- Galeria con composicion limpia y elegante
