# Brief para agente de IA: Marmoleria Villalobos

## Objetivo
Crear una pagina web para Marmoleria Villalobos con una apariencia muy similar a una propuesta previa ya aprobada para una marmoleria. La web debe transmitir elegancia, solidez, confianza y trabajo artesanal de alta calidad.

## Identidad de marca
- Nombre comercial: Marmoleria Villalobos
- Rubro: marmoleria
- Sensacion visual: premium, limpia, fuerte, profesional
- Colores principales:
  - Blanco
  - Negro
  - Naranja

## Recursos disponibles
- Logo disponible actualmente en la raiz del proyecto: [image.png](image.png)
- Carpeta de imagenes del proyecto: [img](img)
- Carpeta de estilos del proyecto: [css](css)

Nota importante: aunque el requerimiento indica que el logo esta en la carpeta img, en el estado actual del proyecto el archivo visible es image.png en la raiz. Si se va a usar en la implementacion, tomar ese archivo como logo principal salvo que el usuario mueva otro recurso a img.

## Direccion creativa
- Mantener una linea visual parecida a una web anterior de marmoleria.
- Evitar estilos genericos o plantillas muy basicas.
- Usar una estetica sobria con acentos naranjas bien medidos.
- Priorizar contrastes fuertes entre blanco y negro.
- El naranja debe usarse para llamadas a la accion, detalles visuales y elementos destacados.

## Estructura recomendada del sitio
1. Encabezado fijo con logo, menu y boton de contacto.
2. Hero principal con mensaje potente, fondo elegante y llamada a la accion.
3. Seccion de servicios.
4. Seccion de materiales o tipos de acabados.
5. Galeria de trabajos o proyectos destacados.
6. Seccion de por que elegirnos.
7. Seccion de contacto con WhatsApp, telefono, direccion y formulario.
8. Pie de pagina con datos de negocio y enlaces rapidos.

## Secciones y contenido sugerido

### Hero
- Titulo sugerido: Transformamos piedra en espacios que perduran
- Subtitulo sugerido: Fabricacion e instalacion de cubiertas, barras, fachadas, lapidas, escaleras y acabados en marmol, granito y cuarzo.
- Botones sugeridos:
  - Solicitar cotizacion
  - Ver trabajos

### Servicios
Incluir servicios como:
- Cubiertas para cocina
- Barras y cubiertas para bano
- Escaleras y pisos
- Fachadas y recubrimientos
- Lapidas y trabajos especiales
- Proyectos residenciales y comerciales

### Materiales
Mostrar categorias o tarjetas como:
- Marmol
- Granito
- Cuarzo
- Piedra sintetica

### Galeria
- Mostrar trabajos en tarjetas o mosaico limpio.
- Resaltar calidad del acabado y detalle visual.
- Si faltan fotos reales, dejar una estructura lista para reemplazo.

### Por que elegirnos
Usar argumentos como:
- Experiencia y precision
- Materiales de alta calidad
- Asesoria personalizada
- Entrega profesional
- Acabados duraderos

### Contacto
Incluir:
- Telefono
- WhatsApp
- Ubicacion
- Horario
- Formulario simple

## Lineamientos de diseno
- Tipografia elegante y con personalidad.
- Fondo claro o secciones alternadas en blanco y negro.
- Botones naranjas con buen contraste.
- Espaciado amplio para que la pagina se vea premium.
- Sombras y bordes sutiles, no recargados.
- Diseno totalmente responsive.

## Lineamientos tecnicos
- Crear una estructura limpia y facil de mantener.
- Separar HTML, CSS y recursos visuales.
- Usar nombres claros para clases y secciones.
- Optimizar imagenes cuando sea posible.
- Preparar el sitio para crecer con mas proyectos y servicios.

## Instruccion clave para el agente
Si no existe el sitio previo como referencia directa, generar una propuesta muy cercana al estilo esperado para una marmoleria premium: elegante, moderna, oscura en contraste y con acentos naranjas energicos, evitando una apariencia corporativa fria o una landing generica.

## Entregable esperado
Una pagina de inicio profesional para Marmoleria Villalobos con estructura visual clara, contenido editable, buena jerarquia tipografica, llamadas a la accion visibles y base lista para agregar imagenes reales y datos finales del negocio.
